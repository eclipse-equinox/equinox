The Equinox project is organized into a number of components or work
areas. Each release cycle (roughly July to June) we plan, implement and
deliver a number of new features as well as updates to existing
function. The pages here capture some of the thinking from the community
in the development of the overall Equinox plan. You can also get to the
more detailed milestone plans for the components via the pages here.

The following links are the official plans for previous and current
releases.

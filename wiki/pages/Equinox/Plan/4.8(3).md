### OSGi R7 implementation

The Photon release of Equinox includes implementations of the OSGi R7
specification for the following OSGi specifications:

  - OSGi Core Framework Specification
  - OSGi Resolver Service Specification
  - OSGi Log Service Specification

All other OSGi specification that Equinox implements remain at OSGi R6
specification levels for the Photon release.
